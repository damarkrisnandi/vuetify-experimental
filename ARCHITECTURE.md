## ğŸ¯ Key Concepts

### 1. Each Dialog is Independent
- Level 1, 2, and 3 are separate Vue components
- Each managed by DialogService with unique ID
- All rendered in DialogContainer

### 2. Data Flows Down, Results Flow Up
- Props flow down: Parent â†’ Level 1 â†’ Level 2 â†’ Level 3
- Results flow up: Level 3 â†’ Level 2 â†’ Level 1 â†’ Parent

### 3. Asynchronous Callbacks
- `dialogRef.onClose(callback)` registers async handler
- Callback executes when dialog closes
- Parent can wait for child result before deciding to close

### 4. Z-Index Management
- Vuetify handles z-index automatically
- Later dialogs appear on top
- Multiple dialogs can coexist
